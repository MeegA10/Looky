LOOkY nació utilizando herramientas como lo es SharePoint, pero, ¿Cómo utilizamos? o ¿Para que lo utilizamos?
Primero, SharePoint es una herramienta de Microsoft donde almacenamos datos.
Para almacenar la información se ocupó Sharepoint en la que se crearon diferentes listas con columnas donde la información necesitada se almacenó según su categoría.

A partir de una de estas lista de Sharepoint aparece una opción de exportar a Power Apps, la cual hace que automáticamente se creen 3 páginas, una donde se crea una lista para elegir la información que se quiere solicitar, otra donde se describe la información detallada y la última para editar la información, estas 3 páginas contienen los datos de la lista que fue exportada. Después desde Power Apps se conectaron las listas faltantes y se crearon sus respectivas páginas (iguales a las 3 anteriores pero con los respectivos datos de cada lista).
